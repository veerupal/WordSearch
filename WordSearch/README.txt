This code is build using cmake 3.20.0-r3 and GnuWin32 tested on DevCPP.
Pre-requisite:
	Install any C++ compiler (DevCPP or MinGW recommended)
	Set the system environment path for the compiler 
To run the exe file follow the below instruction:
1-> Open DOS or Command Prompt
2-> Go to the project directory
3-> Type: 
	WordSearch no_of_arguments arg1 arg2 arg3 arg4 arg5 ...
arg1 arg2 arg3 ... are the input given to search the words in dictionary.
